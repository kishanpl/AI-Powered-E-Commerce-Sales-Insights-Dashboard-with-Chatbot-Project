# AI-Powered E-Commerce Dashboard

### How I met the requirements:
1. **Data Cleaning:** Handled in `app.py` using Pandas to fix dates and column names.
2. **SQL Analysis:** Integrated `DuckDB` directly into the dashboard to run SQL queries on the data.
3. **Power BI Specs:** Replicated all requested KPI cards and charts (Sales, Profit, Orders, Category, Region, Trends) using Plotly for a high-end interactive experience.
4. **AI Chatbot:** Integrated OpenAI's ChatGPT API to allow natural language questions about the sales data.
5. **Filters:** Fully interactive sidebar for Region and Category.
